💼 Sistema de Gerenciamento de Pessoas em Java
📌 Sobre o Projeto

Este projeto é uma aplicação desenvolvida em Java, com o objetivo de praticar e consolidar conceitos de Programação Orientada a Objetos (POO).

O sistema simula o cadastro e gerenciamento de pessoas, incluindo clientes e funcionários, além de suas informações detalhadas como endereço, telefone e profissão.

🚀 Funcionalidades
Cadastro de Clientes
Cadastro de Funcionários
Associação de dados:
📍 Endereço completo
📞 Telefone (DDD, número e tipo)
💼 Profissão e Cargo
Estrutura organizada por pacotes
